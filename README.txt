1) Upload .github/workflows/build-exe.yml to your repo in the same path.
2) Upload package.json and main.js to the repo root.
3) Keep your existing index.html, style.css, app.js, config.js, favicon.ico, favicon.png, apple-touch-icon.png, logo.png.
4) Go to Actions > Build Windows EXE > Run workflow.
5) Download artifacts from the successful run.
