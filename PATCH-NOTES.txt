Pharmacy System - Final Patched Google Sheets Version

Included:
- app.js patched to use Google Sheets Apps Script backend only
- Duplicate warning bypass for patient file number 999444
- Code.gs backend for Google Sheets CRUD tables
- google-sheets-admin-tools.gs with menu tools:
  * Zero All Drug Stocks
  * Clear All Transactions

Required setup:
1) Open your target Google Sheet.
2) Extensions > Apps Script.
3) Paste Code.gs into your Apps Script project.
4) Optionally add google-sheets-admin-tools.gs as a separate script file.
5) Run setupSheets() once.
6) Deploy > New deployment > Web app.
   - Execute as: Me
   - Who has access: Anyone
7) Copy the Web App URL.
8) Paste it into config.js as APPS_SCRIPT_WEB_APP_URL.
9) Publish your frontend files.

Notes:
- If the sheet looked empty before, it was because the previous frontend was not writing to Google Sheets.
- This patched version routes save/read/update/delete through Apps Script.
- The Apps Script project must be attached to the same spreadsheet you want to use as the database.
